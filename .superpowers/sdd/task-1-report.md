## Issues Encountered and Resolutions

1. **Vitest picked up e2e tests**: The initial `include: ['tests/**/*.test.js']` caused Vitest to collect `tests/e2e/smoke.test.js`, which failed because Playwright's `test()` API is not compatible with Vitest. Fixed by scoping include to `tests/unit/**/*.test.js` and adding `exclude: ['tests/e2e/**']`.

2. **http-server not pre-installed**: The `webServer` command uses `npx http-server` which auto-installs on first run (expected behavior for npx). No configuration change needed.
